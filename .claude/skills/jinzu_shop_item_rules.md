# 金珠商店物品使用规则

## 概述
金珠商店（assist标签页）出售的物品有严格的使用上下文限制，不能在背包中随意使用。

## 物品type分类规则

| type | 含义 | 示例 |
|------|------|------|
| `potion` | 药品（恢复类） | 止血草、金疮药、养生丸等 |
| `material` | 材料（种子、任务物品、副将技能书等） | 草药种子、断肠草、连击入门 |
| `chest` | 宝箱/礼包 | 铁质宝匣、1级武器礼盒 |
| `consumable` | 消耗品（银两包、血石/魔石） | 小银两包、小血石 |
| `other` | 其他（秘药、金丹、双倍经验卡、续命灯等） | 攻击秘药、攻击金丹、双倍经验卡 |
| `vip` | VIP物品 | 诸侯令 |
| `quest` | 任务物品 | 吴军密函 |

**重要分类变更**（与旧版不同）：
- 死亡替身符: `other`（非consumable）
- 双倍经验卡: `other`（非consumable）
- 副将补药/魔药/经验丹/招募令: `other`（非potion/material）
- 续命灯: `other`（非potion）
- 催熟剂: `other`（非consumable）
- 免战符: `other`（非material）
- 种子: `material`（非seed）
- 玫瑰花: `material`，`is_usable: false`

## 物品分类与使用方式

### 背包可用物品（is_usable: true）
| 物品 | 效果 |
|------|------|
| 诸侯令1天/7天/30天 | 使用获得对应天数VIP特权 |
| 小银两包 | 使用获得100000银两 |
| 大银两包 | 使用获得500000银两 |
| 免战符 | 使用进入300分钟免战状态（需120分钟内未主动攻击他人） |
| 小/中/大血石 | 使用获得对应点数生命储备（5000/10000/25000） |
| 小/中/大魔石 | 使用获得对应点数魔法储备（5000/10000/25000） |

### 场景消耗物品（is_usable: false，不可背包使用）
| 物品 | 消耗场景 | 规则 |
|------|----------|------|
| 回城符 | 地图→回城→【点击回城】 | 仅城镇地图可用，传送到所在区域驿站 |
| 神行符 | 地图→神行→选择传送点 | 仅城镇地图可用，快速传送到区域内传送点 |
| 活力卡 | 山庄首页→行动力旁【补充】按钮 | 每张恢复10点行动力，上限120，行动力满时不显示按钮 |
| 催熟剂 | 百草园→生长中作物旁【催熟】 | 立即让作物成熟，无催熟剂时不显示按钮 |
| 死亡替身符 | PK死亡时自动消耗 | 免除死亡惩罚（不损失经验/银两/道具/荣誉），每次PK死亡消耗1个 |
| 断肠草 | 离婚流程中消耗 | 离婚必备道具 |
| 结交酒 | 红颜/知己结交流程 | 结交NPC所需 |
| 断交酒 | 红颜/知己断交流程 | 断交NPC所需 |

## 秘药礼包规则
- 使用后**随机获得一种**秘药x1（不是全部）
- 可能获得：攻击秘药、防御秘药、生命秘药、魔法秘药、暴击秘药、闪避秘药
- 使用`random_one_of`效果实现，而非`random_items`

## 物品ID映射
- 回城符: items.json中为`return_scroll`，map_service.py中也使用`return_scroll`
- 神行符: items.json中为`speed_scroll`，map_service.py中也使用`speed_scroll`
- 经验丹种子有两个ID: `seed_jingyandan`(银两购买) 和 `exp_seed`(金珠购买)，SEEDS字典中两者都有定义

## 关键代码位置
- 物品使用逻辑: `services/item_service.py` → `use_item()`
- 地图回城/神行: `services/map_service.py` → `town()` / `shenxing()`
- 百草园种植/采摘/催熟: `services/villa_service.py` → `plant_seed()` / `harvest_plot()` / `ripen_plot()`
- 山庄活力卡: `blueprints/villa.py` → `use_vitality_card()`
- 百草园种子定义: `services/villa_service.py` → `SEEDS` dict

## 秘药状态显示
- 角色查看页(view_player.html)显示"状态:生↑ 攻↑"等
- `PlayerModel.status_effect` property自动根据TempEffect表生成
- 缩写映射: 生=生命上限, 魔=魔法上限, 攻=攻击, 防=防御, 暴=暴击, 闪=闪避, 经=经验
- 无活跃效果时显示"状态:无"
- 其他玩家查看也能看到

## 物品效果提示
- 背包列表中可使用物品旁显示灰色效果提示
- 如 "攻击秘药 攻击+5%" "小血石 生命储备+5000" "诸侯令1天 VIP+1天"
- `DataService.get_item_effect_hint(item_id)` 方法生成摘要文本
