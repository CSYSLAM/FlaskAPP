# 怪物掉落装备规则

## 规则概述

怪物掉落装备遵循等级区间制，根据怪物等级和类型（普通/精英）决定掉落装备的等级范围和品质范围。

## 掉落等级限制规则

- **怪物不会掉落比自己等级低10级以上的装备**
- 例如：21级普通怪物不会掉落青布套（10级装备），因为21-10=11 > 10
- 此规则确保怪物掉落的装备等级与其自身等级保持在合理范围内

## 神器规则

**核心原则：神器与普通装备是两条完全独立的品质体系**

- **非神器装备（`is_artifact: false`）**：无论是怪物掉落、打造、开宝箱，**永远不会产出神器品质**。可产出品质为：普通、精良、卓越、史诗
- **神器装备（`is_artifact: true`）**：**只会产出神器品质**，不会产出普通/精良/卓越/史诗。只有专门配置了神器掉落的怪物、宝箱才能产出
- **普通怪物、精英怪物永远不出神器**，无论装备模板的 `is_artifact` 是什么值
- **打造系统永远不出神器**

简表：

| 来源 | 非神器装备 | 神器装备 |
|------|-----------|---------|
| 普通怪物 | 普通、精良 | 不掉落 |
| 精英怪物 | 精良、卓越、史诗 | 不掉落 |
| 打造 | 精良、卓越、史诗 | 仅神器 |
| 宝箱/活动 | 按配置 | 仅神器 |

## 装备属性设计规则

### 基础属性（base_stats）

- **每件装备最多只有1-2种基础属性**
- **一般装备只有1种基础属性**：
  - 武器 → `attack`（攻击）
  - 防具（头盔/衣服/手套/裤子/鞋子）→ `defense`（防御）
  - 饰品 → `defense`（防御）
- **厉害一点的装备有2种基础属性**（需在模板中明确指定），如：`attack` + `defense`，或 `defense` + `max_health`
- 特殊饰品可以有 `attack` + `defense`（目前未实现，后续添加）

### 附加属性（extra_stats）

- 附加属性在生成装备时随机附加，受品质控制
- 品质与附加属性条数：普通(1条)、精良(2条)、卓越(3条)、史诗(4条)、神器(6条)
- 附加属性从 `max_extra_stats` 中随机选取，数值受星级影响

### 现有装备示例

| 装备 | 部位 | 基础属性 |
|------|------|---------|
| 雏龙长剑 | 武器 | attack |
| 雏龙头巾 | 头盔 | defense、max_health |
| 雏龙衣 | 衣服 | defense、max_health |
| 雏龙手套 | 手套 | attack、crit_rate |
| 雏龙鞋 | 鞋子 | dodge_rate、defense |

## 核心规则

### 等级区间划分

- **1-10级怪物**：掉落1-9级装备
- **11-20级怪物**：掉落10-19级装备
- **21-30级怪物**：掉落20-29级装备
- **31-40级怪物**：掉落30-39级装备
- **41-50级怪物**：掉落40-49级装备
- **以此类推...**

### 普通怪物掉落规则

| 怪物等级区间 | 装备等级范围 | 可掉落品质 |
|-------------|-------------|-----------|
| 1-10级 | 1-9级 | 普通、精良 |
| 11-20级 | 10-19级 | 普通、精良 |
| 21-30级 | 20-29级 | 普通、精良 |
| 31-40级 | 30-39级 | 普通、精良 |
| 41-50级 | 40-49级 | 普通、精良 |
| 以此类推... | 以此类推... | 普通、精良 |

**规则公式**：
- 装备等级范围：`[怪物等级区间下限, 怪物等级区间上限 - 1]`
- 例如：11-20级普通怪物 → 掉落10-19级装备

**掉落模板**：
- 1-10级普通怪：掉落雏龙套（1级全职业通用）和麻布套（5级全职业通用）
- 11-20级普通怪：掉落青布套（10级全职业通用）
- 21级以上普通怪：掉落三职业专属套装混合池（与同等级区间精英怪相同的模板池），品质限制普通、精良

### 精英怪物掉落规则

| 怪物等级区间 | 装备等级范围 | 可掉落品质 | 掉落套装 |
|-------------|-------------|-----------|---------|
| 1-10级 | 1-9级 | 精良、卓越、史诗 | 雏龙套（全职业） |
| 11-20级 | 10-19级 | 精良、卓越、史诗 | 豹皮套(战)、水纹套(术)、苍狼套(刺) |
| 21-30级 | 20-29级 | 精良、卓越、史诗 | 雷霆套(战)、日光套(术)、夜色套(刺) |
| 31-40级 | 30-39级 | 精良、卓越、史诗 | 精钢套(战)、远虑套(术)、疾风套(刺) |
| 41-50级 | 40-49级 | 精良、卓越、史诗 | 百炼套(战)、锦绣套(术)、雀翔套(刺) |
| 51-60级 | 50-59级 | 精良、卓越、史诗 | 黑龙套(战)、凤鸾套(术)、雁翔套(刺) |

**规则公式**：
- 装备等级范围：`[怪物等级区间下限, 怪物等级区间上限 - 1]`
- 精英怪掉落职业专属套装，三个职业的套装混合在掉落池中
- 职业专属装备（class_required 不为 null）只有对应职业的玩家才能装备

### 掉落数量规则

- 每次击败怪物，**只能随机概率掉落一件装备**
- 普通怪物掉落概率：`drop_rate = 0.15`
- 精英怪物掉落概率：`drop_rate = 0.3`
- 具体掉落的装备模板从怪物配置的 `templates` 列表中随机选择

### 品质权重规则

普通怪物的品质权重配置：
```json
{
  "common": 50,    // 普通
  "uncommon": 30,  // 精良
  "rare": 15,      // 卓越
  "epic": 4,       // 史诗
  "legendary": 1   // 神器（概率极低）
}
```

精英怪物的品质权重配置：
```json
{
  "common": 15,    // 普通
  "uncommon": 30,  // 精良
  "rare": 30,      // 卓越
  "epic": 20,      // 史诗
  "legendary": 5   // 神器
}
```

**注意**：虽然配置中包含所有品质的权重，但根据掉落规则，普通怪物实际只应用普通和精良品质。精英怪物实际应用精良、卓越、史诗品质。

## 绑定规则

- **普通怪物**掉落的装备：**绑定**
- **精英怪物**掉落的装备：**非绑定**

## 星级规则

- 装备星级通过 `star_range` 或 `star_weights` 控制
- 默认星级范围：1-5星
- 精英怪物通常有更高概率掉落高星级装备

## 精英怪物材料掉落规则

- **25-40级精英怪**：随机掉落1个材料（碎皮/麻布/黄杨木/黄铜矿中随机一种），掉落概率随等级递增，25级约10%，40级达到33%
- **41-60级精英怪**：随机掉落1个材料（硬皮/棉布/沉香木/黑铁矿中随机一种），掉落概率随等级递增，41级约10%，60级达到33%
- **>60级精英怪**：掉落硬皮/棉布/沉香木/黑铁矿，概率33%

## 装备模板配置

怪物掉落装备需要在 `monsters.json` 中配置 `equipment_drop` 字段：

```json
{
  "equipment_drop": {
    "drop_rate": 0.15,
    "templates": [
      "template_id_1",
      "template_id_2"
    ],
    "rarity_weights": {
      "common": 50,
      "uncommon": 30,
      "rare": 15,
      "epic": 4,
      "legendary": 1
    }
  }
}
```

## 注意事项

1. 装备模板的 `level_required` 必须与怪物等级区间匹配
2. 职业专属装备（class_required 不为 null）只有对应职业的玩家才能装备
3. 全职业装备（class_required 为 null）所有职业均可装备
4. 职业专属套装既可以通过打造系统获得，也可以从对应等级精英怪掉落
5. 后续如需修改规则，需同步更新此文档并检查所有怪物配置
