# 军团领地占领规则

> 整理自 `services/battlefield_service.py` 与 `blueprints/legion.py`，对应游戏内"战场 / 军团领地"玩法。
> 适用于开发参考，亦可作为玩家规则说明底稿。

---

## 一、概述

军团领地占领是军团之间的城池争夺玩法：军团成员进入**战场**击杀敌对方积累**军团积分**，战场结束后由**该城积分最高的军团**占领城池，占领军团的**全体成员**获得属性加成。占领状态每周重置，需重新争夺。

核心数据：
- 积分权重（按城等级）：`TIER_POINTS = {basic:1, mid:2, high:3}`
- 占领加成（按城等级）：`TIER_BONUS = {basic:1, mid:2, high:3}`
- 进入凭证：`TIER_TOKEN = {basic:'battle_flag_1', mid:'battle_flag_2', high:'battle_flag_3'}`

---

## 二、参与资格

- 必须**已加入军团**（无成员身份无法进入战场，也无占领资格）。
- 进入战场需消耗对应等级的**战场令旗**，每次进入扣 1 个：
  - `battle_flag_1`（低级）→ 低级城：成都 / 永安 / 建宁 / 北平 / 晋阳 / 许昌 / 建邺 / 吴郡 / 柴桑
  - `battle_flag_2`（中级）→ 中级城：江陵 / 下邳 / 汉中（中立）
  - `battle_flag_3`（高级）→ 高级城：洛阳（中立）
- **国家锁定**：低级九城仅限**本国玩家**进入；中/高级中立城所有人可进。
- 处于战斗 / PK 中、或已在其它战场内时不可进入。

---

## 三、战场开放时间

- **正式团战**：每周六 **20:00 – 20:30**（`_in_saturday_window`）。
- **测试战**：由工作台触发，持续 **10 分钟**（`TEST_WAR_DURATION = 600`），结束强制清场并按积分自动占领。
- **开发/测试态**：当前 `TESTING_MODE = True`，环境下战场**始终开放**，便于调试。

---

## 四、积分获取规则

击杀对方玩家，或逼使对方在决斗中逃跑判负时，按**城市等级**结算积分：

- **个人战场积分** `personal_battle_points` += 等级分
- **军团积分** `Legion.battle_points`（持久化）+ 内存态 `legion_scores[军团]` += 等级分
- 该城**玩家榜** `player_scores[玩家]` += 等级分
- **同军团互殴 = 友好切磋，不结算任何积分。**
- 击杀**不转移**荣誉 / 银两 / 经验（有意设计，与野外 PK 区分）。
- 逃跑判负：对手获得同等积分（同军团除外）。

---

## 五、占领判定（谁赢下这座城）

占领 / 领取时采用**惰性结算**：取该城当前积分榜**最高的军团**为胜者（`winner_legion_id`）。

- 内存态 `legion_scores` 非空 → 取其中最高军团。
- 内存态为空（跨天 / 重启清零）→ **回退到持久化 `Legion.battle_points` 最高军团**。
- **同城仅允许一个军团占领**：设置归属时会清除其它军团的同名占领（`_set_city_owner`）。

---

## 六、占领操作流程

入口：军团 → **领地页** `/legion/territory`，页面列出"已占领城池"与"可领取城池"。

- 仅**军团长**（`role == 'leader'`）可访问 `/legion/occupy/<city_key>`。
- 成功条件：
  1. 该城胜者是你的军团；
  2. 该城**尚未**被你军团占领。
- 失败提示：
  - 非军团长 → "只有军团长可以占领城市"
  - 本军团非胜者 → "你的军团未赢得该城市"
  - 已占领 → "该城市已被你的军团占领"
- 成功后该城写入军团 `occupied_cities`，**全团成员立即获得属性加成**。

---

## 七、占领加成

按城市等级给全团成员加成（`get_territory_bonuses`）：

- `attack` / `defense` / `max_health` / `max_mana` 各 + 对应 `TIER_BONUS` 值。
- 多座城加成**叠加**。

| 城等级 | 积分权重 | 占领加成（每项） |
|---|---|---|
| 低级 basic | 1 | 1 |
| 中级 mid | 2 | 2 |
| 高级 high | 3 | 3 |

---

## 八、周期重置

- **每周六 0 点**（`ensure_weekly_territory_reset`）：
  - 清空所有军团 `occupied_cities = []`；
  - 清零 `Legion.battle_points` 与 `LegionMember.personal_battle_points`；
  - 重置内存各城积分状态，待本周团战重新争夺。
- 测试战结束 `settle_war / _auto_settle_territories` 也会按当日积分**自动占领**。

---

## 九、端到端流程（玩家视角）

1. 加入军团 → 准备对应等级战场令旗。
2. 战场开放时段进入某城战场（消耗 1 令旗，满血满蓝入场）。
3. 击杀敌国 / 其它军团玩家（或逼其逃跑判负），为自己与军团累积对应等级积分；同军团互殴不计分。
4. 战场结束（周六 20:30 强制清场 / 测试战 10 分钟到时）：按各城军团积分惰性结算出胜者军团。
5. **军团长**在领地页对"可领取"城市执行占领 → 写入 `occupied_cities`，全团立即获得加成。
6. 每周六 0 点重置，重新争夺。

---

## 十、排行榜 与 占领归属 的区别（重要）

战场页展示两个榜，但**只有军团维度决定占领**：

- **玩家榜（前 10）**：来自 `player_scores`，仅用于**展示 / 荣誉**，**不参与**占领判定。
- **军团榜（前 5）**：来自 `legion_scores`（内存），是占领判定依据；内存为空时回退到持久化 `battle_points`。
- 结论：个人再强也只是拉高军团总分；最终"谁占领"看**该城军团积分最高的军团**。

---

## 十一、影响占领先后的其余机制

- **等级分权重**：低级 +1、中级 +2、高级 +3。想高效抢城，优先在**高级城（洛阳）**打出击杀。
- **死亡处理**：被击杀且无 `battle_revive_lamp`（续命灯）→ 立即传出战场；有灯可原地复活继续作战。
- **强制清场**：周六 20:30 或测试战到点，所有在场玩家被传出，随后自动按积分占领。
- **国家锁**：低级九城仅限本国玩家，实际争夺者被国家限制；中/高级中立城才是跨服混战主战场。
- **同场不得跨城**：已在某城战场内，需先离开才能进其它城。

---

## 十二、边界与易错点速查

| 情形 | 行为 |
|---|---|
| 非军团成员 | 不能进战场，更无占领资格 |
| 非军团长点占领 | 拒绝"只有军团长可以占领城市" |
| 本军团不是该城胜者 | 拒绝"你的军团未赢得该城市" |
| 该城已被自己军团占领 | 拒绝"该城市已被你的军团占领"（无重复占领，也无冷却） |
| 内存积分榜因重启清空 | 占领判定回退到 `battle_points` 最高军团 |
| 周六 0 点 | 所有占领与积分清零，需当周重新打 |
| 测试战结束 | 按当日积分自动占领，无需手动 |

---

## 十三、给玩家的规则说明（建议上墙）

- 占领 = 军团荣誉，**全员受益**（加成按城等级叠加）。
- 积分只看**军团总分**，个人榜是面子不是里子。
- 高级城分值最高，是兵家必争之地。
- 每周重置，周日~周五为空窗期，周六团战定归属。
- 同军团互殴为友好切磋，不计积分；击杀不掠夺对方荣誉银两。
